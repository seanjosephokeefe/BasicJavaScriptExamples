Example 02

Say hello.....  

This example uses a div to display either the message with the hello or a warning that nothing was entered in the text box. There is a check to see if the textbox is empty by checking the length of the value of the textbox. If value.length==0 then there is no input.

<h3>Example 02</h3>
<form action="#">
	<p>
		Say hello.....  <input id="TBHello" type="text" />
	</p>
	<div id="DivMessage"></div>
	<input id="BtnHello" type="button" value="Say Hello" onclick="SayHello();" />
	<input id="BtnReset" type="reset" value="Clear" 
	onclick="document.getElementById('DivMessage').innerHTML = '';" />
</form>

<script type="text/javascript">
function SayHello() {
	var myTB = document.getElementById('TBHello').value;
	var myDiv = document.getElementById('DivMessage');
	if (myTB.length == 0)
		message = "Please enter the name for someone to say hello to.";
	else
		message = "Hello " + myTB + ".";
	myDiv.innerHTML = message;
}	
</script>
